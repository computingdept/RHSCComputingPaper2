2.2 Programming Fundamentals SIS

Programming is like giving your computer a recipe to follow. You write instructions in a language similar to English, one step at a time. This step-by-step plan is called Sequence. Sometimes, the program must make a choice (that’s Selection), or repeat the same steps over and over (that’s Iteration). We also use Variables to store information, and different Data Types (like words, numbers, or yes/no answers) tell the computer what kind of information it’s dealing with.

Key concepts (SIS): Sequence (the order of instructions), Iteration (repeating actions), Selection (making decisions).

2.3 Producing Robust Programs

Good programs are reliable and secure. Techniques include validation (check input is correct) and authentication (password protection).

Errors can be syntax errors (typos) or logic errors (wrong results).

2.4 Boolean Logic

Computers use logic gates to make decisions: AND, OR, and NOT.

Example: In an AND gate, both inputs must be 1 to get an output of 1.

2.5 Programming Languages & IDEs

Computers understand different types of languages: high-level (Python) and low-level (Assembly).

Programs are translated using a compiler (fast) or interpreter (easy to debug).
