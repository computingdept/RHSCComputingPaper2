String Operations & Casting Trace Table Challenges

These sequential challenges involve various string operations (length, substring extraction, left/right, concatenation, case conversion) and type‐casting operations (casting numbers to strings and vice versa, as well as ASCII conversions). Fill in the trace table only when a variable’s value changes or when output is produced.

ST1: String Length

Pseudocode:

01 subject = "ComputerScience"
02 length = subject.length
03 print("Length is " + length)
Line Number subject length Output

ST2: Substring Extraction

Pseudocode:

01 subject = "ComputerScience"
02 subStr = subject.substring(3,5)
03 print("Substring is " + subStr)
Line Number subject subStr Output

ST3: Left Substring

Pseudocode:

01 subject = "ComputerScience"
02 leftPart = subject.left(4)
03 print("Left part is " + leftPart)
Line Number subject leftPart Output

ST4: Right Substring

Pseudocode:

01 subject = "ComputerScience"
02 rightPart = subject.right(3)
03 print("Right part is " + rightPart)
Line Number subject rightPart Output

ST5: Concatenation of Two Strings

Pseudocode:

01 stringA = "Hello, "
02 stringB = "World"
03 result = stringA + stringB
04 print(result)
Line Number stringA stringB result Output

ST6: Concatenation with Literal

01 name = "Alice"
