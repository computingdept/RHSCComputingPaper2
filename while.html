Passcode Trace Table Challenge

Your program asks a user to enter a passcode until the correct one is entered. The correct passcode is J277. For example, the user’s first attempt is Computer, the second is Science, and the third is J277. Then the program prints the total number of guesses. 

// First attempt "Computer"
// Second attempt "Science"
// Third attempt "J277"
01 correctcode = "J277"
02 exit = false
03 attempts = 0
04 while (exit == false)
05    attempts = attempts + 1
06    password = input("Enter passcode:")
07    if password == correctcode then 
08         exit = true
09    else 
10         print("Incorrect")
11    endif
12 endwhile
13 print( attempts + " guesses")

Fill in the table below with the correct values for each line.
