Q1: Passcode (do...until)

Scenario: A locker passcode “J277” is required. The user tries “Computer,” “Incorrect,” “Science,” “Incorrect,” then “J277.” The program prints the total attempts.

01 correctcode = "J277"
02 exit = false
03 attempts = 0
04 do
05    attempts = attempts + 1
06    password = input("Enter passcode:")
07    if password == correctcode then
08       exit = true
09    else
10       print("Incorrect")
11    endif
12 until (exit == false)
13 print(attempts + " guesses")
