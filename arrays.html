Array Trace Table Challenges

These 10 sequential challenges demonstrate basic operations with arrays—including 1D and 2D arrays. In each example, arrays are declared (or assigned initial values), elements are accessed or updated, and output is printed. Remember that arrays are 0‑indexed and can store only one data type.

Example A1: Declare and Assign a 1D Array

Pseudocode:

01 array colours[5]
02 colours = ["Blue", "Pink", "Green", "Yellow", "Red"]
03 print("colours declared")
Line Number colours Output

Example A2: Print an Element from a 1D Array

Pseudocode:

01 array colours[5] = ["Blue", "Pink", "Green", "Yellow", "Red"]
02 print(colours[2])
Line Number colours Output

Example A3: Change an Element in a 1D Array

Pseudocode:

01 array colours[5] = ["Blue", "Pink", "Green", "Yellow", "Red"]
02 colours[3] = "Noni"
03 print("New colour: " + colours[3])
Line Number colours Output

Example A4: 2D Array Assignment and Access

Pseudocode:

01 array gameboard[8,8]
02 gameboard[1,0] = "Pawn"
03 print("Position (1,0): " + gameboard[1,0])
Line Number gameboard Output

Example A5: Assign to an Element in a Names Array

Pseudocode:

01 array names[5]
02 names[3] = "Noni"
03 print("names[3] is " + names[3])
Line Number names Output

Example A6: Sum of Elements from a 1D Array

Pseudocode:

01 array numbers[4] = [2, 4, 6, 8]
02 sum = numbers[0] + numbers[1] + numbers[2] + numbers[3]
03 print("Sum is " + sum)
Line Number numbers sum Output

Example A7: 2D Array – Corner Pieces on a Board

Pseudocode:

01 array board[3,3]
02 board[0,0] = "Rook"
03 board[2,2] = "King"
04 print("Corner pieces: " + board[0,0] + " and " + board[2,2])
Line Number board Output

Example A8: Concatenation of Array Elements

Pseudocode:

01 array colours[5] = ["Blue", "Pink", "Green", "Yellow", "Red"]
02 message = colours[0] + ", " + colours[1] + ", " + colours[2]
03 print("Message: " + message)
Line Number colours message Output

Example A9: Replace an Element in a 2D Array

Pseudocode:

01 array board[4,4]
02 board[1,1] = "Knight"
03 board[1,1] = "Bishop"
04 print("Piece at (1,1): " + board[1,1])
Line Number board Output

Example A10: Update a Pre-assigned 1D Array

Pseudocode:

01 array fruits[3] = ["Apple", "Banana", "Cherry"]
02 fruits[1] = "Orange"
03 print("Fruits: " + fruits[0] + ", " + fruits[1] + ", " + fruits[2])
Line Number fruits Output
