Sequence Trace Table Challenges

These examples involve only sequential (straight‐line) execution—assignments and output statements (no selection or iteration).

Example 1: Add Two Numbers First number is 5 and secondnumber is 3

Pseudocode:

01 a = int(input("First number"))
02 b = int(input("Second number"))
03 sum = a + b
04 print("Sum is " + sum)

Record the trace only when a value changes or output is produced.

Line Number a b sum Output
