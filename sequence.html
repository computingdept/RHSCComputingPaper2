Sequence Trace Table Challenges

These challenges show how a program runs line by line in SEQUENCE—without any SELECTION (like “if” or switch/case) or ITERATION (for/while loops). Each line happens in order, and we keep track of the values and outputs at each step.

Example 1: A local café is developing a self-service till system that allows customers to enter the quantity of two different items they are purchasing. In one example, a customer enters 5 for the first item and 3 for the second item.

Pseudocode:

01 a = int(input("First number"))
02 b = int(input("Second number"))
03 sum = a + b
04 print("Sum is " + sum)

Record the trace only when a value changes or an output is produced.

Line Number a b sum Output
