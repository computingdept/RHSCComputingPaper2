Q5: Array Element Search

Scenario: In an English class, a student searches for the word "apple" in the array of fruits: ["banana", "orange", "apple", "kiwi", "grape"].

Pseudocode:

01 fruits = ["banana", "orange", "apple", "kiwi", "grape"]
02 found = false
03 i = 0
04 while (i < 5 and found == false)
05     if fruits[i] == "apple" then
06         found = true
07     endif
08     i = i + 1
09 endwhile
10 print("Found apple: " + found)
Line Number fruits found Output

Pseudocode:

01 clubs = ["Drama", "Music", "Art", "Debate", "Science"]
02 i = 0
03 j = 4
04 do
05     temp = clubs[i]
06     clubs[i] = clubs[j]
07     clubs[j] = temp
08     i = i + 1
09     j = j - 1
10 while (i < j)
11 print(clubs)
Line Number clubs Output
