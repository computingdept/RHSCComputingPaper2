1D Array Trace Table Challenges

Below are 10 trace table challenges related to 1D arrays. For lines that produce an actual print, you only need to type the raw output (e.g., Green instead of Printed: Green). For lines that do not produce output, a short descriptive message (e.g., scores assigned) may still appear if you want to show what changed. Feel free to remove those if you prefer an empty cell.

Q1: Basic Array Access

Scenario: In an art class, a teacher has prepared a palette of colours: ["Blue", "Pink", "Green", "Yellow", "Red"]. The program prints the colour at index 2.

01 colours = ["Blue", "Pink", "Green", "Yellow", "Red"]
02 print(colours[2])

Q5: Array Element Search

Scenario: In an English class, a student searches for the word "apple" in the array of fruits: ["banana", "orange", "apple", "kiwi", "grape"].

01 fruits = ["banana", "orange", "apple", "kiwi", "grape"]
02 found = false
03 i = 0
04 while (i < 5 and found == false)
05     if fruits[i] == "apple" then
06         found = true
07     endif
08     i = i + 1
09 endwhile
10 print("Found apple: " + found)
